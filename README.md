Student Marks Manager

A basic online tool that tracks pupils' scores or results.

Features
- Put in the student's name along with their score
- Auto mark tallying
Show students one by one in rows using a grid layout
- Simple and clean UI
- No sign-in, also no storage behind

Technologies Used
- HTML
- CSS
- JavaScript

Live Demo
student-marks-manager.vercel.app

How to Run Locally
1. Download the project
2. Launch index.html using a web browser of your choice
